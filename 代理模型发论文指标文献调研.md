# 代理模型测试指标与论文发表可行性文献调研

生成日期：2026-05-31

## 1. 结论

结构抗震响应代理模型没有统一的“达到某个指标即可发表”的硬性标准。是否能发论文，通常取决于三件事：

1. 模型是否解决了一个明确的工程问题，例如减少 OpenSees 非线性时程分析成本、预测最大层间位移角、支持阻尼器布置优化等。
2. 测试指标是否达到同类文献的常见水平，并且在未见地震波、未见结构参数组合上具有泛化能力。
3. 是否有足够的方法学贡献，例如多模态输入、物理可解释性、尾部风险识别、消融实验、与传统 ML/DL 模型对比等。

结合目前文献水平，建议将指标分为三个层级：

| 评价等级 | R2 | MAE（层间位移角原始比值） | RMSE | MAPE | 适合的论文定位 |
|---|---:|---:|---:|---:|---|
| 基本可投 | >= 0.85 | <= 0.0015-0.0020 | <= 0.0025-0.0040 | <= 25% | 方法有特色、工程应用型论文 |
| 比较稳妥 | >= 0.90 | <= 0.0010 | <= 0.0015-0.0020 | <= 20% | 普通期刊/工程应用论文较有说服力 |
| 强结果 | >= 0.94-0.96 | <= 0.0007 | <= 0.0010-0.0015 | <= 10%-15% | 可主打高精度代理模型 |

对当前模型的判断：

当前最好已测试版本已经具备“可以写论文”的基础，但不建议只靠精度作为唯一卖点。更合适的论文定位是：

> 基于地震波小波时频图、结构标量参数与阻尼器布置特征的多模态代理模型，用于快速预测钢框架/阻尼结构最大层间位移角。

如果目标是 `Engineering Structures`、`Structures`、`Journal of Building Engineering` 等结构工程方向期刊，并且主打预测精度，建议继续把测试集 R2 推到 `>= 0.90`，同时改善高漂移尾部样本的识别能力。

## 4. 指标解释与写论文时的注意点

### 4.1 不要只看 MAPE

当前模型 MAPE 约 `19%`，看起来不算非常低，但这不一定代表模型很差。原因是层间位移角存在大量接近 0 的样本，MAPE 的分母是真实值。当真实值很小时，即使绝对误差很小，相对误差也会被放大。

Hyndman 和 Koehler (2006) 指出，许多常用预测误差指标在常见情形下会退化，并建议采用更稳健的误差评价方式。对当前任务，建议论文中同时报告：

1. R2
2. MAE
3. RMSE
4. MAPE 或 SMAPE
5. 分区间 MAE/RMSE
6. 高漂移阈值的 Precision、Recall、F1
7. 预测-真实散点图
8. 残差分布图
9. 按楼层数、阻尼器布置、地震波强度分组的误差箱线图

### 4.2 需要区分“平均响应预测”和“尾部风险预测”

当前模型在 `MIDR >= 0.005` 时 F1 已达到 `0.916`，说明对中等漂移样本识别较好。但 `MIDR >= 0.010` 后 F1 下降到 `0.541`，说明模型对强非线性或危险尾部样本仍有漏报问题。

如果论文要讨论结构安全、损伤状态、屈服判断或危险预警，尾部指标必须单独报告，不能只给全局 R2 和 MAE。

### 4.3 建议增加基线对比

为了增强论文说服力，建议至少加入以下基线模型：

1. XGBoost / LightGBM / CatBoost，仅用结构参数 + 地震动强度指标。
2. MLP，仅用标量特征。
3. 1D-CNN / LSTM / Transformer，直接用地震波时程或频谱特征。
4. 当前 2D-CNN 多模态模型。
5. 消融实验：无小波图、无阻尼器布置、无地震波派生特征、无 FiLM/门控融合。

如果当前模型在 R2 上不是最高，也可以从以下角度构造贡献：

1. 比传统 ML 更能利用地震波时频特征。
2. 在未知地震波测试集上泛化更稳定。
3. 对阻尼器布置变化有更好的适应性。
4. 在计算成本上显著低于 OpenSees NLTHA。
5. 提供可解释性分析，例如特征重要性、Grad-CAM、SHAP、频带敏感性。

## 5. 推荐发表门槛

若目标是普通中文核心、会议、或工程应用型英文期刊：

```text
R2 >= 0.85
MAE <= 0.0010-0.0015
RMSE <= 0.0020
MAPE <= 25%
并且有明确工程应用价值
```

当前模型基本达到这一层级。

若目标是较稳妥的英文期刊论文：

```text
R2 >= 0.90
MAE <= 0.0010
RMSE <= 0.0015-0.0020
MAPE <= 20%
MIDR >= 0.010 的 F1 >= 0.75
```

当前模型的 MAE、RMSE、MAPE 接近或达到，但 R2 和尾部 F1 仍建议提升。

若目标是以“高精度代理模型”为主要创新点：

```text
R2 >= 0.94
MAE <= 0.0007
RMSE <= 0.0010-0.0015
MAPE <= 10%-15%
MIDR >= 0.010 的 Recall/F1 >= 0.85
```

当前模型尚未达到这一层级。
